You may have more than one Java installed, for example the Minecraft launcher tends to install Java 1.12, but we need to use Java 1.8 for Evocraft. You don't need to remove anything, just make sure to use the right version in the right place. To list the currently installed versions, use 

```
update-java-alternatives --list
``` 

and look for the one that contains `java-1.8.0`. That's the one we'll need to launch the Evocraft server. For example, on my machine launching Evocraft looks like:

```
/usr/lib/jvm/java-1.8.0-openjdk-amd64/bin/java -jar spongevanilla-1.12.2-7.3.0.jar
```

You'll also need to make sure to use the same Java version if you are starting Minecraft from the launcher. In the Minecraft launcher under installations, select your v1.12.2 installation, click on the ellipsis symbol and choose options. In the options screen, choose "MORE OPTIONS", where you should see the option to specify the Java executable. You'll want to paste the same path as used above to launch Evocraft into that box.
